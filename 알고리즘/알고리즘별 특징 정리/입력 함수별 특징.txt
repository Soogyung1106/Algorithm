※ 공백 있는 문자열 입력 받을 때 -> cin 사용시 맨 앞 글자만 입력됨. 
(엔터나 공백을 기준으로 끊어 받는 입출력 함수이므로)

<입력 함수들의 특징>
scanf -> 공백이 있는 문자열 입력 x ('엔터' 혹은 '공백'을 종료로 받아들임.)
fgets -> 공백이 있는 문자열 입력 o 

cin -> 공백이 있는 문자열 입력 x ('엔터' 혹은 '공백'을 종료로 받아들임.)
getline -> 공백이 있는 문자열 입력 o ('엔터'를 종료로 받아들임.)

<상황별 사용해야 하는 입력함수>
(상황1) 숫자 혹은 문자를 띄어쓰기 없이 일열로 입력 받지만, 한개씩 끊어받아야 할 경우?
→ scanf("%1d", &x), scanf("%c", &x)
♧왜(Why)♧ scanf는 엔터 혹은 공백(스페이스바)를 기준으로 끊어 읽기 때문!!

(상황2) 공백이 있는 문자열을 입력 받을 경우?
→ getline(cin, string); 
♧주의사항♧ 버퍼에 엔터가 남아있을 수 있기 때문에 비워줘야 함.!
